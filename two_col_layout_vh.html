Two Col Layout with Viewport Units

Here are instructions on as how to create a two col layout with equal height columns with Viewport Units.

Your HTML structure should look like this: 


                        <div class="container">
                            <aside></aside>
                            <main></main>
                        </div>

And your CSS should look like this: 


                    body, container, aside, main {
                        height: 100vh; <!-- Give body, the columns and their container 100 % viewport height -->
                    }
                    aside {
                        float: left; <!-- Floats the sidebar left -->
                    }
                    main {
                        overflow: auto; <!-- Gives the main section scrollbars if need be -->
                    }

More detailed instructions can be found in https://stanhub.com/how-to-make-div-element-100-height-of-browser-window-using-css-only/
